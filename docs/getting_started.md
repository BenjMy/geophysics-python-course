# Getting Started

## First Steps

1. **Launch Jupyter Lab**:
   ```bash
   jupyter lab
   ```

2. **Navigate to notebooks**:
   - Open `notebooks/01_introduction/01_python_basics.ipynb`

3. **Run your first notebook**:
   - Press `Shift + Enter` to run cells
